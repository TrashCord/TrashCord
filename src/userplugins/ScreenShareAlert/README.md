# Extension-ScreenShareAlert

ScreenShare Alert is a plugin for Vencord that automatically detects Screen Share, webcam, and possible external recordings in Discord calls. It features modern notifications, customizable colors, an anti-spam system, Drag & Drop, and only works while you are in a call.

---

# ScreenShare Alert

A modern plugin for Vencord developed to monitor activities in Discord voice calls in real time.

**ScreenShare Alert** automatically detects when someone starts screen sharing, turns on their camera, or when a possible external recording is identified during the call, displaying elegant and fully customizable notifications on screen.

Designed with a focus on performance, practicality, and a modern look, the plugin provides a clean and intuitive experience without interfering with your conversation.

---

## ✨ Features

* 🖥️ Automatic Screen Share detection
* 📹 Webcam and video stream detection
* 🔴 Experimental system for detecting external recordings
* 📞 Only works while you are in a call
* 🚫 Option to ignore your own sharing
* 🎨 Full customization of notification colors
* 📍 Choose notification position on screen
* ✋ Drag & Drop system to freely move alerts
* ⚡ Fast, lightweight, and animated notifications
* 🧠 Intelligent anti-spam system
* 🖼️ Displays avatar and username of the detected user
* 🌈 Custom gradients for each alert type

---

## 🎨 Customization

The plugin allows you to configure:

* Screen Share alert colors
* Video alert colors
* Recording alert colors
* Notification size
* Custom height and width
* Distance from edges
* Notification position
* Free movement via drag and drop

All directly in the plugin settings.

## 📥 Installation Tutorial

### 1. Open the Vencord folder

Example:

```txt
C:\Users\YourName\Documents\Vencord
```

---

### 2. Enter the folder:

```txt
src
```

Then:

```txt
plugins
```

---

### 3. Create a folder called:

```txt
ScreenShareAlert
```

---

### 4. Drag the `index.tsx` file into the folder

The final path should look like:

```txt
Vencord/src/plugins/ScreenShareAlert/index.tsx
```

---

### 5. Open CMD inside the Vencord folder

In the Explorer address bar (where the folder path appears):

```txt
Documents > Vencord
```

Double-click until the text turns blue, type:

```txt
cmd
```

and press ENTER.

---

### 6. Build the plugin

Make sure CMD opens like this:

```txt
C:\Users\YourName\Documents\Vencord>
```

Then type:

```bash
pnpm build
```

and press ENTER.

---

### 7. Restart Discord

After building, restart Discord completely.

---

### 8. Enable the plugin

Open:

```txt
Vencord Settings > Plugins
```

Search for:

```txt
Screen
```

Enable the plugin and you're done ✅

---

## 🚀 Goal

The goal of ScreenShare Alert is to provide a modern and efficient visual system to quickly alert you when important activities happen inside your call, keeping you informed in real time without needing to manually monitor Discord's interface.
