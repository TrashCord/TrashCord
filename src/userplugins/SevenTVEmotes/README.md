# SevenTVEmotes

A custom [Vencord](https://vencord.dev/) user plugin. Adds a 7tv emote picker to easily send 7tv emotes through discord.

## Installation

It is required to build Vencord to use custom plugins. Follow this guide for getting set up: https://docs.vencord.dev/installing/custom-plugins/

In a terminal inside the `userplugins` folder, run:

```bash
git clone https://github.com/lolspers/SevenTVEmotes.git
```

Then rebuild Vencord using

```bash
pnpm build
```

> [!NOTE]
> User plugins have to be manually updated.
> To update the plugin, run `git pull` in the `userplugins/SevenTVEmotes` directory.
